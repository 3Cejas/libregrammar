Description of Arabic tag sets
=============================
The arabic tag sytax for grammar check is composed of 4 parts

The tag starts by word class
for example 
V: Verb
N: Noun
P: tool or partical

Nouns
======

4 parts:
======
* Prefix
* Lexem
* Suffix
* Ecletic

Prefix: 
=======
Noun prefixes :
=============
contains 3 parts
W: conjonction: starts by WAW or FEH, take 3 values: W: for waw, F; for Feh, -: none.
B: preposition: B: for Beh, K: for Kaf, L: for Lam, by default we use B, -: none.
L: definite article, L: for AL,  -: none.

example:
وبالأكل =WBL
بأكل: -B-
والأكل: W-L

Lexem:
======
 define the attributes of word in dictionary, for example, a word can be signle in dictionary, but can have pluarl suffix which change its number.
contains 4 parts
- Word type:
	N: noun,
	V: verb,
	P: particle
- Word sub class:
	M: Masdr
	A: Adjective
	...
	-: not defined
- Gender:
	F: feminin
	M: masculine
	-: None
- Number:
	1: single
	3: plural, in this case, plural in dictionary which means irregular pluarl (جمع تكسير)


example:
مدرسة: N-F1
مدرس:N-M1
مدراس: N-M3

Suffix:Noun conjugation  :
======
	- Gender suffix:
		M: masculine ( ون is a mark of masculine)
		F: feminine (ة or ات)
		-: none
	- Number suffix:
		1: single
		2: dual
		3: plural ( regular masculine or feminine plural)
	- Case ( إعراب)
		R: marfou3
		J: Majzoum
		N: Mansoub
		B: Mabni
	Person: 
		H: 3rd person : always
Ecletic
======
define the extended words added to the lexem: الضمائر المضافة

	Ponoun:
		Person: 
			I: 1st person
			Y: 2nd person
			H: 3rd person
		gender: 
			M/F
		Number: 1/2/3


Verbs
=====
Prefixes + lexem + conjugation + ecletic 
Verb prefixes :
=============
	W: conjonction: starts by WAW or FEH, take 3 values: W: for waw, F; for Feh, -: none.
	S: future prefix, س+يتعلم

verb Lexem  :
=============
	Word type: V
	verb class:
		W: weak verb
		Wa: ajwaf
		Wn: naqis
		Wm: mithal
		D: doubled مضعّف
	weak letter origin;
		Y: yeh weak letter
		W: waw weak letter

	transitivity: 
		T: transitivity/ commun, 
		I: intransitivity
	length: verb length 
	3-6: indicate length

verb conjugation  :
=============
	tense:
		P: past
		F: present
		I:  Imperative
	voice:
		P: passive
		A: active
	case:
		R: marfou3
		J: Majzoum
		N: Mansoub
		B: Mabni
	Ponoun:
		Person: 
			I: 1st person
			Y: 2nd person
			H: 3rd person
		gender: 
			M/F
		Number: 1/2/3

		Person:

			H: 3rd person
		gender:
